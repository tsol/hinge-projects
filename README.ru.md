## в чём суть

знаешь чувство, когда приходит идея и хочется быстро забабахать прототип прямо с телефона лёжа на диване? ну вот.

**HINGE** — это крошечный Vite плагин, который внедряет в твой фронтенд-проект оверлей. тыкаешь на шестерёнку, пишешь что хочешь изменить, и AI-агент правит код — HMR подхватывает и изменения тут же появляются в браузере. никакой IDE, никакого терминала, даже за стол садиться не надо.

ты запускаешь **cloudflared туннель** до своего локального дев-сервера, получаешь публичный URL, открываешь его на телефоне — и вайбкодишь откуда угодно.

установка — одна команда. буквально **одна**:

```
https://raw.githubusercontent.com/tsol/hinge-projects/main/skills/hinge-projects-install.md
```

кидаешь ссылку агенту. готово.

---

## что получаешь

- дев-сервер с Hinge панелью на любой твоей машине
- публичный URL — доступ с телефона, планшета, откуда угодно
- **создание новых Vue/React проектов** с уже встроенным Hinge — одна команда
- агент знает структуру проекта, читает/пишет файлы, всё живое в браузере
- можешь даже дать доступ нетехническому другу — он сможет оставлять задачи в очереди

## как использовать из Telegram (или любого чата)

когда всё запущено, просто говоришь агенту:

| Ты говоришь | Агент отвечает |
|-------------|----------------|
| `чё работает?` | `hinge → https://xxx.trycloudflare.com ✅` + список проектов |
| `какой урл у krollo?` | `https://plasma-sun-jon-motors.trycloudflare.com` |
| `создай вью проект test-app` | `✅ Created test-app … url=https://...` |
| `создай реакт блог test-blog` | `✅ Created blog … url=https://...` |
| `удали test-app` | `✅ Removed test-app` |
| `перезапусти krollo` | `✅ Restarted, tunnel: https://...` |

агент внутри использует `hip` — то же самое что в терминале:

```bash
hip                          # список проектов + статус + URLs
hip hinge status             # детальный статус hinge
hip krollo start             # запуск дев + туннель
hip krollo restart           # перезапуск, туннель остаётся
hip create vue test-app --start
hip create react blog --start
hip delete test-app --force
```

---

## команды

| Команда | Что делает |
|---------|------------|
| `hip` | Список проектов со статусом и ссылками |
| `hip <name> start` | Запустить дев-сервер + туннель |
| `hip <name> stop` | Остановить дев-сервер + туннель |
| `hip <name> restart` | Перезапустить; сохранить туннель |
| `hip <name> status` | Детальный статус |
| `hip create vue <name> [--start]` | Создать Vue 3 + TS + Hinge |
| `hip create react <name> [--start]` | Создать React + TS + Hinge |
| `hip create list` | Доступные фреймворки |
| `hip delete <name> --force` | Удалить проект |

## ⚠️ братан, серьёзно

**hinge-projects создаёт публичные Cloudflare туннели прямо в твой локальный дев-сервер.**

- любой у кого есть URL может открыть твоё приложение и Hinge панель
- **никакой аутентификации** — нет логина, пароля, токена
- это строго для **личного прототипирования и вайбкодинга**
- **не** направляй это на продакшен, базы данных, криптокошельки или бабушкин рецепт печенья

я предупредил. используй на свой страх и риск.
